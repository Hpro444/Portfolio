C (C11)pthreadsPOSIX TrieLock-CouplingSystems
View Details
Key Features
  • Thread-Safe Trie: Per-node mutex locks with hand-over-hand (lock-coupling) locking for concurrent inserts and traversals
  • Background Scanner Threads: Up to 10 threads poll directories every 5 s, detecting new or modified files via stat() timestamps
  • Live Search Thread: Prints all words matching a user prefix once per second, concurrent with ongoing file ingestion
  • Clean Shutdown: pthread_cancel + pthread_join with full trie memory reclamation
Impact
  • Demonstrates low-level concurrency — lock-coupling eliminates global lock contention
  • Handles live file system changes without interrupting search queries
